API

This details the entire public API of Wyam, including all official modules. The API is roughly comprised of three namespaces:

  • Wyam.Common: This is a common set of interfaces, abstractions, and helper classes. It is commonly used by module developers and is also available from within your configuration file.
  • Wyam.Core: This contains the actual implementations for most of the Wyam infrastructure. It also contains the implementations of core modules that don't require any external dependencies. It is generally only used in embedded scenarios. Module authors shouldn't need to include or use these objects.
  • Wyam.Modules: These namespaces contain implementations of the various non-core modules that require external dependencies. These namespaces are automatically brought in scope in the configuration file if the module is loaded.
