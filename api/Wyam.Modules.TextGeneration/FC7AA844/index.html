GenerateContent Class

Namespace
Wyam.Modules.TextGeneration
Interfaces

Summary

Procedurally generates content using flexible Rant text templates.

Syntax

public class GenerateContent : RantModule, IModule

Constructors

Name Summary
GenerateContent(ContextConfig) Uses a function to determine a text template which is processed and added as content for each document. This allows you to specify different content for each document depending on the context.
GenerateContent(DocumentConfig) Uses a function to determine a text template which is processed and added as content for each document. This allows you to specify different content for each document depending on the input.
GenerateContent(IModule[]) The specified modules are executed against an empty initial document and the resulting content from evaluating the entire child module chain is processed as a text template and added as content to each input document.
GenerateContent(object) The specified text template is processed and added as content for every input document.
