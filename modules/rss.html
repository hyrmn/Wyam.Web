Rss

Creates RSS feed from input documents.

Implements RSS and Atom feeds. All input documents should have RssTitle and RssDescription metadata because they are required by RSS. You can override these metadata keys via WithTitleMetaKey(string key) and WithDescriptionMetaKey(string key).

This module outputs input files without changes and one another file: the RSS feed. The RSS feed contains RelativeFilePath and IsRssFeed: true metadata keys. Use WriteFiles module inside Branch module without parameters to place file in appropriate file system location. See example.

Options:

  • Use WithoutGuid() to disable generation of unique identifier for each input document. Guid is generated using input document's source location. You may want to disable it if your data is generated on fly.
  • Optional RssPubDate metadata will be used to specify document's publication date. You can override this metadata key with WithPublicationDateMetaKey(string key).
  • Language can be set for feed using WithLanguage(string lang). Language string must follow these conventions: http://www.rssboard.org/rss-language-codes
  • Use WithLinkCustomizer to create pretty urls.

Some things to know:

  • If publication date for each document is not set, most RSS reader programs will assume that most recent publication is the top most in the output RSS file (that is first input document).
  • Use OrderBy module to order your documents in right order.
  • Use a feed validator when generating RSS for first time: https://validator.w3.org/feed/
  • Only basic tags are defined in this module at this moment, eg. you cannot attach author tag to specify RSS item's author.

Examples

Input/posts/hello.md: 

    ---
    RssTitle: Hello, world!
    RssDescription: My first blog post
    RssPubDate: 10/11/12
    ---
    
    Hello, RSS world!
config.wyam: 

    Pipelines.Add("Blog posts",
        ReadFiles("posts/*.md"),
        FrontMatter(Yaml()),
        Markdown(),
        WriteFiles(".html"),
        Rss(siteRoot: "http://example.org",
            outputRssFilePath: "posts/feed.rss",
            feedTitle: "My awesome blog",
            feedDescription: "Blog about something"
        ),
        WriteFiles()
    );
Output/posts/feed.rss: 

    <![CDATA[<?xml version="1.0" encoding="utf-8"?>
    <rss version="2.0" xmlns:atom="http://www.w3.org/2005/Atom">
      <channel>
        <title>My awesome blog</title>
        <description>Blog about something</description>
        <link>http://example.org</link>
        <atom:link href=" http://example.org/posts/feed.rss" rel="self" />
        <lastBuildDate>Wed, 30 Dec 2015 13:39:16 +0300</lastBuildDate>
        <item>
          <title>Hello, world!</title>
          <link>http://example.org/posts/hello.html</link>
          <description>My first blog post</description>
          <pubDate>Sat, 10 Nov 2012 00:00:00 +0400</pubDate>
          <guid isPermaLink="false"> 81a26807-355a-dbf9-7729-c6601f1d8a2b</guid>
        </item>
      </channel>
    </rss>

Usage

  • Rss(string siteUri, FilePath rssPath, string feedTitle, string feedDescription)

    Creates RSS feed from input documents.

    • siteUri

      The full URI of the site, including hostname (example: "http://mysite.com/blog/").

    • rssPath

      Relative output location where generated RSS feed will be placed on server. (example: "feeds/feed.rss").

    • feedTitle

      Title of the RSS feed.

    • feedDescription

      Description of the RSS feed.

  • Rss(Uri siteUri, FilePath rssPath, string feedTitle, string feedDescription)

    Creates RSS feed from input documents.

    • siteUri

      The full URI of the site, including hostname (example: "http://mysite.com/blog/").

    • rssPath

      Relative output location where generated RSS feed will be placed on server. (example: "feeds/feed.rss").

    • feedTitle

      Title of the RSS feed.

    • feedDescription

      Description of the RSS feed.

Fluent Methods

Chain these methods together after the constructor to modify behavior.

  • AssumePermalinks()

    Assume that RSS item link is unique and permanent.

  • WithDescriptionMetaKey(string key)

    Set metadata key to lookup in documents for RSS item description.

  • WithGuidMetaKey(string key)

    Set metadata key to lookup in documents for RSS item Guid.

  • WithLanguage(string lang)

    Set feed language.

    • lang

      RSS language code

  • WithLinkCustomizer(Func<FilePath, FilePath> customizer)

    Allows to customize output links. Does not execute when document explicitly defines output URL via RssItemUrl metadata.

    • customizer

      A delegate that takes and returns a Wyam.Common.IO.FilePath.

  • WithoutGuid()

    Do not append GUIDs to RSS items.

  • WithPublicationDateMetaKey(string key)

    Set metadata key to lookup in documents for RSS item publication date.

  • WithTitleMetaKey(string key)

    Set metadata key to lookup in documents for RSS item title.

Metadata

The following metadata is added to each document.

  • RelativeFilePath: FilePath

    Relative path to the output RSS file.

  • WritePath: FilePath

    Relative path to the output RSS file (primarily to support the WriteFiles module).
