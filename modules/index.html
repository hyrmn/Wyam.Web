Modules

Content
Append Appends the specified content to the existing content of each document.
AutoLink Wyam.Modules.Html Replaces occurrences of specified strings with HTML links.
Content Replaces the content of each input document with the string value of the specified content object.
CsvToHtml Wyam.Modules.Tables Converts CSV content to HTML tables.
ExcelToCsv Wyam.Modules.Tables Transforms Excel content to CSV.
GenerateCloudSearchData Wyam.Modules.AmazonWebServices Generates bulk JSON upload data to get documents into Amazon CloudSearch.
GenerateContent Wyam.Modules.TextGeneration Procedurally generates content using flexible Rant text templates.
GenerateJson Wyam.Modules.Json Converts objects stored in metadata or elsewhere to JSON.
HtmlEscape Wyam.Modules.Html Automatically escapes HTML content.
Image Wyam.Modules.Images This module manipulates images by applying a variety of operations.
MinifyCss Wyam.Modules.Minification Minifies the CSS content.
MinifyHtml Wyam.Modules.Minification Minifies the HTML content.
MinifyJs Wyam.Modules.Minification Minifies the JS content.
MinifyXhtml Wyam.Modules.Minification Minifies the XHTML content.
MinifyXml Wyam.Modules.Minification Minifies the XML content.
Prepend Prepends the specified content to the existing content of each document.
Replace Replaces a search string in the content of each input document with new content.
ReplaceIn Replaces a search string in the specified content with the content of input documents.
Rss Creates RSS feed from input documents.
SearchIndex Wyam.Modules.SearchIndex Generates a JavaScript-based search index from the input documents.
Sitemap Generates a sitemap from the input documents.
Control
Branch Evaluates the specified modules with each input document as the initial document and then outputs the original input documents without modification.
Combine Combines all of the input documents into a single output document.
Concat Executes modules and concatenates their output with the input documents.
ConcatBranch Executes a sequence of modules against the input documents and concatenates their results and the original input. This is similar to Branch except that the results of the specified modules are concatenated with the original input documents instead of being forgotten.
Documents Inserts documents into the current pipeline.
ForEach Executes the input documents one at a time against the specified child modules.
FrontMatter Extracts the first part of content for each document and sends it to a child module for processing.
GroupBy Splits a sequence of documents into groups based on a specified key function.
If Evaluates a series of child modules for each input document if a specified condition is met.
Merge Replaces the content and merges metadata of each input document with the results of specified modules.
ModuleCollection Executes child modules enabling better reuse.
OrderBy Orders the input documents based on the specified key function.
Paginate Splits a sequence of documents into multiple pages.
Switch Executes sequences of modules depending on whether the input document contains a specified value.
Take Takes the first X documents from the current pipeline and discards the rest.
Where Filters the current sequence of modules using a predicate.
Extensibility
Execute Executes custom code for each input document.
Trace Outputs trace messages during execution.
Input/Output
CopyFiles Copies the content of files from one path on to another path.
Download Downloads from HTTP and outputs the results as new documents.
ReadApplicationInput Reads text provided to the application on startup.
ReadFiles Reads the content of files from the file system into the content of new documents.
ReadProject Wyam.Modules.CodeAnalysis Reads all the source files from a specified msbuild project. This module will be executed once and input documents will be ignored if a search path is specified. Otherwise, if a delegate is specified the module will be executed once per input document and the resulting output documents will be aggregated. Note that this requires the MSBuild tools to be installed (included with Visual Studio).
ReadSolution Wyam.Modules.CodeAnalysis Reads all the source files from a specified msbuild solution. This module will be executed once and input documents will be ignored if a search path is specified. Otherwise, if a delegate is specified the module will be executed once per input document and the resulting output documents will be aggregated. Note that this requires the MSBuild tools to be installed (included with Visual Studio).
UnwrittenFiles Outputs only those documents that have not yet been written to the file system.
WriteFiles Writes the content of each input document to the file system.
Metadata
AnalyzeCSharp Wyam.Modules.CodeAnalysis Performs static code analysis on the input documents, outputting a new document for each symbol.
CopyMeta Copies the specified meta key to a new meta key, with an optional format argument.
DirectoryMeta Applies metadata from specified input documents to all input documents based on a directory hierarchy.
Excerpt Wyam.Modules.Html Finds the first occurrence of a specified HTML element and stores it's contents as metadata.
FileName Optimizes a specified metadata key as a filename.
GenerateMeta Wyam.Modules.TextGeneration Procedurally generates metadata using flexible Rant text templates.
GitCommits Wyam.Modules.Git Outputs documents and metadata for commits in a Git repository.
GitContributors Wyam.Modules.Git Outputs documents and metadata for contributors in a Git repository.
GitHub Wyam.Modules.GitHub Outputs metadata for information from GitHub.
HtmlQuery Wyam.Modules.Html Queries HTML content and creates new documents with content and metadata from the results.
Index Adds a one-based index to every document as metadata.
Json Wyam.Modules.Json Parses JSON content for each input document and stores the result in it's metadata.
Meta Adds the specified metadata to each input document.
ValidateMeta<T> Tests metadata for existence, typing, and supplied assertions.
Xmp Wyam.Modules.Xmp Reads XMP data from the input documents and adds it to the document metadata.
Yaml Wyam.Modules.Yaml Parses YAML content for each input document and stores the result in it's metadata.
Templates
Less Wyam.Modules.Less Compiles Less CSS files to CSS stylesheets.
Markdown Wyam.Modules.Markdown Parses markdown content and renders it to HTML.
Razor Wyam.Modules.Razor Parses, compiles, and renders Razor templates.
Xslt Transforms input documents using a supplied XSLT template.
